HARDEN-SEC is a Python-based security auditing tool for Linux systems. 
It automates the detection of common misconfigurations, evaluates system risk, and generates 
professional reports in multiple formats. It is designed for learning, lab environments, 
and as a starting point for practical Linux hardening.

Usage
Run HARDEN-SEC as root for a complete audit:

sudo python3 main.py --format txt
sudo python3 main.py --format json
sudo python3 main.py --format html

Reports are saved in the reports/ directory
